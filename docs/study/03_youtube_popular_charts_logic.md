# 🔝 유튜브 뮤직 대한민국 인기 차트 연동 (Study Guide 03)

현재 유튜브 뮤직에서 가장 사랑받는 100곡(Top 100)을 우리 웹사이트에서도 실시간으로 확인하고 재생할 수 있게 구현합니다.

---

## 🎯 1. 학습 목표

*   **API 없이도** 공신력 있는 데이터를 가져오는 영리한 선택법을 배웁니다.
*   **외부 라이브러리(`yt-search`)**를 활용하여 데이터 구조를 다루는 법을 익힙니다.

## 🛠️ 2. 왜 플레이리스트 ID를 사용하는가?

유튜브 차트(`charts.youtube.com/kr`)는 표면적으로는 단순한 웹 페이지이지만, 내부적으로는 유튜브 뮤직의 공식 플레이리스트와 실시간으로 동기화되어 있습니다.

*   **Playlist ID:** `PL4fGSI1pDJn6jXS_Tv_N9B8Z0HTRVJE0m` (대한민국 인기곡 TOP 100)
*   이 ID 하나만 있으면, 복잡한 API 키 발급 없이도 곡 제목, 아티스트, 영상 ID를 대량으로 가져올 수 있습니다. (구글이 공식적으로 노출하는 데이터이므로 안전합니다.)

## 🚀 3. 라이브러리: `yt-search`

유튜브 데이터 API는 하루에 요청할 수 있는 양(Quota)이 매우 작고, 복잡한 설정이 필요합니다.  
하지만 **`yt-search`** 라이브러리는 가볍게 필요한 정보만 빠르게 검색하거나 플레이리스트의 목록을 싹 긁어올 수 있게 해줍니다.

### 데이터 구조 (예시)
```javascript
const list = await yts({ listId: '...' });
// list.videos 에 아래와 같은 배열이 담깁니다:
[
  {
    title: "뉴진스 (NewJeans) 'Ditto' Official MV",
    videoId: "pSUydWEqKwE",
    author: { name: "NewJeans" },
    thumbnail: "..."
  },
  ...
]
```

## 🔄 4. 탭(Tabs) 레이아웃의 구조

기존에는 홈 화면이 하나뿐이었지만, 이제는 두 가지 세상이 공존합니다.

1.  **Curated (추천):** 우리 사용자들이 정성껏 올린 DB(`supabase`) 기반의 음악들.
2.  **Popular (인기):** 유튜브 서버에서 실시간으로 긁어온 글로벌 인기 차트 음악들.

이를 위해 **Radix UI(또는 Shadcn)**의 `Tabs` 컴포넌트를 사용하여, 사용자가 버튼을 눌러 부드럽게 화면을 전환할 수 있도록 설계합니다.

## ✨ 5. 팁: 플레이어 통합

'추천' 음악이나 '인기' 음악이나 모두 결국 유튜브 영상입니다.  
하나의 **공통 플레이어(GlobalPlayer)**로 재생될 수 있도록 설계했기 때문에, 차트의 음악을 눌러도 하단 플레이어 바에서 즉시 재생되는 멋진 경험을 제공할 수 있습니다.
